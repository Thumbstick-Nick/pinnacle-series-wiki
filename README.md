# Pinnacle Series Wiki

A shared internal reference for how Pinnacle Series actually behaves — what's where, who can do what, and the small details that trip people up.

## Audience

Sales, CSM, Support, QA, and new hires at Eagle Point Software. Use this as a quick lookup when you need to answer "what permissions does someone need to do X?" or "where is feature Y in the Admin Portal?"

## Live wiki

Once GitHub Pages is enabled (Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: / (root) → Save), the wiki will be live at:

**https://thumbstick-nick.github.io/pinnacle-series-wiki/**

## What's in this repo

- `index.html` — the wiki itself. Open it directly to view, or visit the GitHub Pages URL above.
- `screenshots/web/` — the screenshots embedded by the wiki.

## Maintenance

The canonical source of truth for the facts in this wiki is `Pinnacle_UI_Facts.md` (maintained by the QA team, not in this repo). When new facts are captured there, the wiki is regenerated and re-published here.
